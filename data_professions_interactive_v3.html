Архитектура Данных

Разбираемся в фундаменте: чем озеро отличается от хранилища и зачем нужен гибридный подход.

🗄️ Data Warehouse

Хранилище данных. Структурированные данные, очищенные и готовые к отчетам. Используется бизнес-аналитиками.

  • Схема "перед записью" (Schema-on-Write)
  • Высокая скорость SQL-запросов
  • Примеры: Snowflake, Google BigQuery, Oracle Exadata

🌊 Data Lake

Озеро данных. Сырые данные любого формата (текст, фото, логи). Хранится всё подряд для будущего анализа.

  • Схема "при чтении" (Schema-on-Read)
  • Дешевое хранение (S3, HDFS)
  • Примеры: AWS S3, Azure Data Lake Storage, MinIO

🏠 Lakehouse

Гибрид (Озеро + Дом). Современный стандарт. Сочетает дешевизну озера и надежность хранилища.

  • Поддержка транзакций (ACID)
  • Технологии: Delta Lake, Apache Iceberg
  • Единая платформа для ML и BI

🆘 Как не захлебнуться в озере?

Озеро данных без управления превращается в "болото". Основные проблемы и решения:

  • Проблема: Непонятно, какие данные актуальны.
    Решение: Data Catalog и метаданные.
  • Проблема: Низкое качество данных ("мусор на входе").
    Решение: Data Quality проверки на пайплайнах.
  • Проблема: Безопасность доступа.
    Решение: Role-Based Access Control (RBAC).

Гайд: Старт без опыта

Как пробиться в индустрию, если в вакансиях требуют 3 года опыта? Делаем Пет-проекты!

1. Pet-проект "С нуля"

Собери свой мини-пайплайн.

  • Возьми данные с открытого портала (например, данные о погоде или курсах валют через API).
  • Напиши скрипт на Python, который забирает их каждый час.
  • Сохраняй в базу данных (PostgreSQL) или CSV.
  • Построй простой график в Excel или PowerBI.

2. Портфолио на GitHub

Код должен быть виден.

  • Залей код проекта на GitHub.
  • Напиши хороший README: "Что делает проект", "Как запустить", "Какой стек использован".
  • Работодатель смотрит не на диплом, а на код.

3. Стажировки

Входная точка в большие компании.

  • Следи за программами стажировок (Яндекс, Сбер, Тинькофф, стартапы).
  • Даже неоплачиваемая стажировка 2 месяца даст строчку в резюме "Коммерческий опыт".

4. Hard Skills Минимум

Что учить в первую очередь?

  • SQL: Обязательно (Select, Join, Group By).
  • Python: Базовый синтаксис, библиотеки pandas.
  • Git: Уметь сохранять версии кода.
