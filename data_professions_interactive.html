Карьерное Дерево (Career Path)

Start
Junior Data Engineer / Analyst

Написание простых запросов, поддержка существующих пайплайнов, работа с Excel/SQL.

➜
Middle
Data Engineer / BI Developer

Проектирование схем данных, оптимизация запросов, создание дашбордов, работа с облаками.

➜
Senior
Senior DE / Data Architect

Архитектура больших систем, выбор стека технологий, менторство, безопасность данных.

➜
Lead / C-Level
CTO / Head of Data

Стратегия развития данных в компании, управление командами, бюджетирование.

1. Что вам нравится делать больше всего?

Исследовать данные, искать закономерности, строить красивые графики
Строить надежные системы, настраивать серверы, писать код для автоматизации
Придумывать алгоритмы, обучать нейросети, предсказывать будущее
